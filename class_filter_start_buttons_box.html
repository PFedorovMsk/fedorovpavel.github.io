§ computeSizes()

void FilterStartButtonsBox::computeSizes ( )
private

Вычисляет размеры для виджетов.

Это, в частности, необходимо при изменении шрифтов, так как размеры элементов зависят от них.

Предупреждения
В Linux и Windows всё работает как задумано, в MacOS - фигово. Косяк либо тут, либо в FontManager.
