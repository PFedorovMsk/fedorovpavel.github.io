Вычисляет и записывает результаты для времени .

n- результат записывается в ячейку с этим индексом.
copy- если false (по-умолчанию), то вычисляются все характеристики, если true, то только характеристики для , остальные копируются из предыдущей ячейки (опция для дискретных фильтров - "порожки" на графиках).
